"use client";

Export `useCreatePoll()` mutation hook with:

Interface `CreatePollInput`:
```typescript
export interface CreatePollInput {
  title: string;
  description: string;
  options: string[];
  expirationBlocks: bigint; // number of blocks until expiration
}
```

The `mutationFn` should:
1. Validate metadata: call `validatePollMetadata({ title, description, options })` — throw on error
2. Compute metadata hash: `await computeMetadataHash({ title, description, options })`
3. Get contract address: `getContractAddress()`
4. Assemble providers: `await assembleMidnightProviders(providers)`
5. Get witness inputs: secret key + current block number
6. Create witnesses
7. Determine current block: `getCurrentBlockNumber(providers.indexerConfig.indexerUri)`
8. Calculate expiration: `currentBlock + input.expirationBlocks`
9. **If no contract address:** Deploy via `deployPollContract(midnightProviders, witnesses)`, then call `callCreatePoll` on the deployed contract
10. **If contract address exists:** Find via `findPollContract(midnightProviders, contractAddress, witnesses)`, then call `callCreatePoll`
11. Call `callCreatePoll(contract, { metadataHash, optionCount: options.length, pollType: PollType.public_poll, expirationBlock })`
12. Extract poll ID from the result (it returns `Uint8Array`)
13. Store metadata off-chain via `fetch("/api/polls/metadata", { method: "POST", body: JSON.stringify({ pollId: bytesToHex(pollIdBytes), metadata: { title, description, options, createdAt: new Date().toISOString() }, metadataHash: bytesToHex(metadataHash) }) })`
14. Return `{ pollId: bytesToHex(pollIdBytes) }`

On success (`onSuccess`): invalidate poll list queries via `queryClient.invalidateQueries({ queryKey: pollKeys.lists() })`.

Import `PollType` dynamically from `@/contracts/managed/contract` (or use the re-export from `@/lib/midnight/contract-types` — check if `PollTypeEnum` works as a runtime value, not just a type).
