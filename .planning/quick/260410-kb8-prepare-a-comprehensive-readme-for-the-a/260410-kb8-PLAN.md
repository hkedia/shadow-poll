6. **Prerequisites**
   - Bun 1.0+ (for package management and runtime)

7. **Local Setup Instructions**
   - Step 1: Clone the repository
   - Step 2: Install dependencies (`bun install`)
   - Step 3: Set up environment variables:
     ```
     DATABASE_URL=postgresql://user:password@host/database
     ```
   - Step 4: Start the API server (`bun run dev:api`)
   - Step 5: Start the frontend (`bun run dev`)
   - Step 6: Open http://localhost:5173 in browser

8. **Testing Setup**
   - Framework: Vitest
   - Commands:
     - `bun test` or `bun run test` — Run all tests once
     - `bun run test:watch` — Run tests in watch mode
   - Testing libraries:
     - Vitest (test runner)
     - @testing-library/react (component testing)
     - @testing-library/jest-dom (DOM matchers)
     - @testing-library/user-event (user interaction simulation)
   - Test structure: All tests in top-level `tests/` folder
   - Test utilities: `tests/lib/` contains shared test helpers

9. **Contract Compilation & Deployment**
   - Compile: `bun run compile:contracts` — Compiles Compact contracts
   - Deploy: `bun run deploy` — Deploys to Midnight Preview network
   - Scripts location: `scripts/deploy-poll.mjs`, `scripts/print-address.mjs`

10. **Key Commands Reference**
    ```
    bun install        # Install dependencies
    bun run dev        # Start Vite dev server (port 5173)
    bun run dev:api    # Start API server (port 3001)
    bun run dev:all    # Start both servers concurrently
    bun run build      # Production build
    bun run serve      # Production server
    bun run lint       # ESLint check
    bun test           # Run tests
    ```
