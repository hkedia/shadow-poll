<objective>
Remove the wallet autoconnect feature so that every page reload requires the user to manually click Connect.

Purpose: Autoconnect silently re-establishes wallet sessions on reload, which undermines the UX expectation that users explicitly opt-in to connecting their wallet. Removing it simplifies the wallet flow and eliminates confusing "connecting without user action" states.

Output: All autoconnect logic removed — no localStorage flag, no triggerAutoConnect, no WalletAutoConnect component, no isAutoConnecting state.
</objective>
